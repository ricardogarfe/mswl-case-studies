Instalación
------------

.. note::

    Falta como iniciar el cliente.

Ubuntu::

    * sudo apt-get install mysql-client
    * sudo apt-get install mysql-server

Command Line client
--------------------

Entrar en el client de la base de datos::

    bash> mysql

Crear una base de datos::

    mysql> create database [nombre_de_bbdd];

Acceder a ella::

    mysql> use [nombre_de_bbdd];

Eliminar la tabla::

    mysql> drop [nombre_de_bbdd];

Mostrar tablas::

    mysql> show tables;

Descripción de una tabla::

    mysql> desc [nombre_de_tabla];
